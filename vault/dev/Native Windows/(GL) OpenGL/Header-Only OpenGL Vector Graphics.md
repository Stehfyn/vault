# Header-Only OpenGL Vector Graphics

`nitro-gl` is a header-only C++11 vector graphics library for OpenGL/OpenGL ES. Its niche is small embeddable 2D drawing without pulling in a larger UI renderer: paths, fills, strokes, gradients, and rasterization logic that can run across GL versions.

Compare it with NanoVG. NanoVG is the established immediate-mode 2D UI/vector layer; nitro-gl is attractive when dependency size and header-only integration matter more than ecosystem maturity.
