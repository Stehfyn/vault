# Modern OpenGL Sample Demos

`jgl_demos` is a small modern-OpenGL sample collection. Its value is showing the post-fixed-function baseline: shaders, VAOs/VBOs, textures, transforms, and simple scene structure without the Windows-specific WGL ceremony dominating the sample.

Use it after context creation is solved. For Windows integration questions, pair it with `OpenGL in MFC CView (KB Q127071).md` or `WGL Context Attribute Helper (wglarb).md`; for broader API comparison, pair it with `OpenGL Samples Pack (g-truc).md`.
