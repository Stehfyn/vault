# OpenGL Samples Pack (g-truc)

The OpenGL Samples Pack is a broad conformance-style sample set covering GL versions, extensions, buffer/texture patterns, shader stages, and pipeline features. Its value is breadth and API specificity: each sample tends to isolate one feature more cleanly than a demo app or tutorial series.

Use it as a reference when you need the exact shape of a GL feature in code, then return to the Windows-specific notes for context creation, swap behavior, and interop. It pairs naturally with GLM because both come from the same g-truc ecosystem and both assume you care about GL's conventions explicitly.
