# xbox-live-api — Official Xbox Live Bindings

Microsoft's official C++ and WinRT bindings for Xbox Live services (multiplayer sessions, achievements, social graph, presence, leaderboards, Title Storage). Ships as both a flat-C/COM API and a `winrt::Microsoft::Xbox::Services` projection. Usable from Win32 desktop apps on Windows 10+ once you've signed up with ID@Xbox or Creators Program for a TitleId.
