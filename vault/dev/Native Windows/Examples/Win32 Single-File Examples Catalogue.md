# Win32 Single-File Examples Catalogue

A large (~hundreds) catalogue of single-file C++17 samples, one per Win32 control or concept: every common-control style, every dialog item, OpenGL/Direct2D bring-up, printing, drag-and-drop, etc. Pedagogical rather than production: each example is self-contained with no shared framework, which makes it good for "how do I create *just* an up-down control" lookups and bad for learning idiomatic app structure. Comparable in spirit to Microsoft's old Win32 SDK samples but maintained, C++17, and CMake-built.

Code contribution: use it for isolated calls such as `CreateWindowEx`, `InitCommonControlsEx`, dialog procedures, `WM_NOTIFY`, WGL/D2D bring-up, and print/drag-drop setup.
