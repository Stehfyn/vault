# ComView

ComView is an open-source COM object viewer in the same problem space as OleView: enumerate registered classes, inspect CLSIDs/ProgIDs/threading models, and explore interfaces exposed by COM servers. Its value is practical registry-to-runtime mapping for an API surface that otherwise hides behind activation strings and `IUnknown`.

Use it beside COM and shell-extension entries. Shell namespace extensions, app managers, preview handlers, and old control-panel surfaces are all easier to reason about when you can see the registered COM class, server path, apartment model, and implemented interfaces.
