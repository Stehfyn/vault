# old-new-win32api - Raymond Chen Topic Index

`mity/old-new-win32api` is a topic index for Raymond Chen's Old New Thing posts. Its value is retrieval: Chen often explains the actual behavioral contract behind APIs where the formal docs list parameters but omit the historical reason. The index groups posts by Win32 area such as processes, synchronization, DLLs, COM, windows, messages, and GDI.

Use it when an API behaves "wrong" but consistently: non-client sizing, message reentrancy, handle lifetime, clipping regions, foreground activation, and GDI object ownership are exactly the categories where an Old New Thing post often provides the missing invariant.
