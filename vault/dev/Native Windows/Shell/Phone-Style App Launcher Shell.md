# Phone-Style App Launcher Shell

Replaces the Windows desktop with a phone-style home-screen grid: large icons in a paginated swipeable layout, optimized for touch convertibles (Surface Pro, etc.) where the standard Start menu is fiddly. Enumerates installed Win32 apps and UWP packages via the Shell namespace, lays them out in a configurable grid, and (per typical kiosk-mode patterns) can be set as the shell via the `Shell` registry value or as a startup app over the regular desktop.
