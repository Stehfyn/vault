# Title Bar Customization

WinCenterTitle centers title text by injecting into DWM; CaptionButtonFixer restores normal-sized caption buttons in fullscreen.

## Snippet
> The application injects into DWM and hooks *DrawText* method that is used to compute the area, and draw the title bar text for the window.
