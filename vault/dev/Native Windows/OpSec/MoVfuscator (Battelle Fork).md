# MoVfuscator (Battelle Fork)

Battelle's fork of MoVfuscator is useful as a maintained-ish variant of the original mov-only compiler research. It keeps the same core idea, but the fork is easier to read when studying how an academic novelty gets adjusted for larger programs, tests, and adjacent deobfuscation tooling.

The connection to Windows OpSec is indirect but real: it demonstrates how far instruction selection can be pushed before ordinary disassembly heuristics stop being helpful. Compare it with the original `MoVfuscator` entry and with metamorphic compiler notes.
