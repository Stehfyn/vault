# Xbox ATG Game Dev Samples

Xbox ATG samples are broader than their console branding suggests: Direct3D 11/12, XAudio2, gamepad input, storage, networking, achievements, and system-feature samples that often build as UWP or desktop-adjacent Windows code. The best reason to keep them here is the infrastructure code: device-resource wrappers, swap-chain/device-lost handling, HDR paths, input abstraction, and robust sample organization.

Connect this to DirectX and WinRT rather than to "Xbox" alone. Many Windows graphics samples show a single rendering feature; ATG samples show the surrounding app loop and platform services that games actually need.

Code contribution: use the samples for `DeviceResources`, swap-chain/device-lost handling, XAudio2 setup, gamepad input, HDR paths, and storage/platform-service wrappers.
