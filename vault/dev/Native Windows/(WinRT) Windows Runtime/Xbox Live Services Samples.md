# Xbox Live Services Samples

These samples demonstrate Xbox Live Services flows from client apps: sign-in and XSTS identity, profile/social presence, achievements, Title Storage, multiplayer session directory, and related service calls. The technical interest for Native Windows is the boundary between local WinRT/UWP identity APIs and networked Xbox service APIs.

Keep this beside Game Bar and ATG entries. ATG samples teach graphics/input/audio structure; Xbox Live samples teach account/service integration and the async-heavy shape of modern Windows gaming platform code.

Code contribution: inspect sign-in, `XSTS`, achievements, presence, Title Storage, and multiplayer calls as async service boundaries rather than local Win32 primitives.
