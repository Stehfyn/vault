# MS Learn Win32 API Reference Index

The Win32 API index is the stable starting point for official function, structure, enum, and header documentation. The `/windows/win32/api/<header>/` URL pattern matters because many pages are generated from SDK metadata and are easier to find by header than by product-area landing page.

Use this as the citation anchor for header stub entries. A vault note should still add judgment: which subsystem owns the API, what version/manifest/package restrictions matter, and what adjacent APIs complete the pattern.
