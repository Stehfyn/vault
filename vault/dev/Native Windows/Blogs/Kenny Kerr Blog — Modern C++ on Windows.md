# Kenny Kerr Blog - Modern C++ on Windows

Kenny Kerr's blog repo belongs here because it is one of the few sustained sources on modern C++ over COM, WinRT, Direct2D, DirectComposition, and Windows App SDK internals. The writing is valuable when the SDK reference tells you the ABI but not the C++ ownership model.

Use it for `winrt::implements`, `com_ptr`, delegate lifetime, weak references, composition visuals, and Direct2D/DirectWrite patterns. It connects strongly to Custom Frame acrylic/composition entries and any COM/WinRT interop note.
