# winxp-editors — Archive of Retro Windows Editors

Preservation archive of small Windows text editors that still run on XP / 2000 / 9x: PSPad, EditPlus 3, EmEditor 6, Notepad2, Notepad3 (pre-Notepad4), Crimson Editor, ConTEXT, etc. Useful when working on retro hardware or VMs where modern Electron editors require unavailable runtimes.

```text
Each subfolder contains a self-contained installer or portable zip and
notes minimum OS version + last-working build. Many editors here
shipped UPX-packed and run in <8 MB RAM.
```
