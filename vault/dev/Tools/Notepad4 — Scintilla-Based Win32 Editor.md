# Notepad4 — Scintilla-Based Win32 Editor

Modern continuation of Notepad2/Notepad2++ by Florian Balmer. Scintilla-based Win32 editor with syntax highlighting and code folding for ~70 languages, regex find/replace, multi-edit, optional matepath file-browser pane. Native Win32 binary (no .NET, no Electron), <2 MB, runs everywhere Scintilla does.

```text
notepad4.exe path\to\file.cpp /g 120     # open file at line 120
notepad4.exe /pos:cx=1200,cy=900,x=100,y=100
notepad4.exe /encoding:utf8 /eolmode:lf
```
