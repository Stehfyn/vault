# WinMerge — Windows Diff and Merge Tool

Long-running open-source Windows diff/merge tool. Three-way folder + file compare with syntax highlighting (Scintilla), shell integration, plugin support for custom diff transforms (e.g. compare two `.docx` by extracting text), and patch-file generation. The default freeware diff on Windows for users who don't want Beyond Compare's price tag.

```text
# Command-line: diff two folders, then auto-close on no-difference.
WinMergeU.exe /r /e /u /x "C:\old" "C:\new"
#  /r recursive  /e ESC closes  /u no MRU  /x close on identical
```
