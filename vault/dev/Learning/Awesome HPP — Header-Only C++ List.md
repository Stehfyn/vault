# Awesome HPP — Header-Only C++ List

Pranav Srinivas Kumar's curated index, deliberately restricted to single-header or header-only C++ libraries — the kind you can drop into a project with a single `#include` and zero build-system surgery. Categorised similarly to awesome-cpp (algorithms, args, async, CLI, config, graphics, math, parsing, testing, etc.) but with a much higher signal-to-noise ratio because the header-only constraint filters out most over-engineered projects. Pranav is himself the author of several entries on the list (argparse, indicators, tabulate, csv2), which gives you a sense of the bar.

```text
Useful when: you need a feature now, can't add a vcpkg/conan dep, or are
writing a header-only library yourself and want to see how others structure
inline functions, templates, and the single-header amalgamation step.
```
