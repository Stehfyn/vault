# TheAlgorithms — C++ Reference Implementations

Part of TheAlgorithms organization: a curated, community-contributed corpus of every classic algorithm reimplemented in C++. Each file is self-contained, documented, and CI-tested. Useful as a quick reference when you remember "I know there's a name for this dynamic-programming pattern..." - search the repo, find the canonical C++ impl. Doesn't replace CLRS but pairs well with it.

```text
# Layout (each file is one algorithm, with a Doxygen comment block):
algorithms/       # generic algorithms (sort, search, ...)
math/             # number theory, combinatorics, FFT, etc.
data_structures/  # trees, heaps, union-find, segment trees, ...
graph/            # bfs, dfs, dijkstra, max-flow, MST, ...
dynamic_programming/  # knapsack, lcs, edit-distance, matrix-chain, ...
strings/          # KMP, Z-algorithm, suffix arrays, ...
```
