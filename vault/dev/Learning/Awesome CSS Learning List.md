# Awesome CSS Learning List

Deliberately small, opinionated CSS learning list maintained by Micromata — the editorial position is that "awesome-css" style mega-lists are useless because everything is on them, so this one only admits resources the maintainers actually recommend. Covers the essentials (MDN, css-tricks, Wes Bos's CSS Grid course, Kevin Powell's YouTube channel, Smashing Magazine), modern layout (flexbox/grid), and selected playgrounds. If you already know modern CSS this list won't surprise you; its value is as a "where do I send a new hire" link.

```text
Pair with: the MDN CSS reference (the primary source), web.dev's Learn CSS,
and 1-2 hands-on resources (Flexbox Froggy, Grid Garden, CSS Battle) for
muscle memory.
```
