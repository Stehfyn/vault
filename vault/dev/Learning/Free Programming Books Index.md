# Free Programming Books Index

The canonical megalist of free-to-read programming books, cheat sheets, courses, podcasts, and interactive tutorials — split per language and per subject, then again per natural language. Maintained by the EbookFoundation org with aggressive PR review, so dead links and pirated content get pruned faster than on most "awesome-X" lists. The quality range is enormous: world-class texts (SICP, Crafting Interpreters, The Rust Programming Language, Eloquent JavaScript) sit next to 2003 PHP tutorials, so use it as an index and verify each entry's currency before committing time.
