# Angular Practice Challenges

Curated set of Angular practice exercises - each challenge is a small repo with a broken / incomplete component you need to make pass. Topics range across signals, RxJS operators, change detection (OnPush), forms (template-driven vs reactive), and standalone components. Useful only if you need Angular drills.

```text
# Each subdirectory is a standalone Angular project:
#   1-signals-basics/        # convert legacy @Input bindings to signal()
#   2-rxjs-debounce/         # implement search-as-you-type
#   3-onpush-pitfalls/       # find the missing markForCheck()
# Open the README in each for the exercise spec; run `ng serve` to iterate.
```
