# Awesome GPUI List

Curated index of projects built with GPUI, the Rust GPU-accelerated UI framework that powers the Zed editor. Useful for two things: surveying the (still small) ecosystem of non-Zed GPUI apps to see how the framework is being stretched, and finding example code for patterns the official docs gloss over (entity model, view composition, async tasks, rendering quirks). Not a tutorial — it presupposes you've already read the GPUI source or Zed's own usage.

```text
Browse for: third-party widget libraries, sample apps, GPUI-on-Windows ports,
and writeups on the framework's entity/cx model. Cross-reference with the
zed-industries/zed repo, which remains the largest body of real GPUI code.
```
