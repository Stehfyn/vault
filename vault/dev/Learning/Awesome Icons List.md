# Awesome Icons List

Curated index of icon fonts and SVG icon sets usable on the web: Font Awesome, Material Icons, Heroicons, Phosphor, Lucide, Tabler, Bootstrap Icons, Feather, Remix, Ionicons, plus the usual specialised sets (logos, weather, devicons). Each entry is just a link; the value is the breadth and the inclusion of less-discoverable sets you wouldn't find by searching "icon font." Verify licence on each before shipping — most are MIT/SIL-OFL but a few corporate sets are CC-BY-NC or have brand-asset restrictions.

```text
Decision axes when choosing: licence, sprite vs. font vs. inline SVG,
tree-shakeability (lucide-react vs. font-awesome blob), visual consistency
across the set, and whether the project is still maintained.
```
