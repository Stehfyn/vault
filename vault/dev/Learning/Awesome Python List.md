# Awesome Python List

The original and largest "awesome-list" for Python — a hand-curated, category-organized index of frameworks, libraries, tools, and learning resources. Categories cover web frameworks, async, ORMs, CLIs, GUIs, ML, data viz, packaging, deployment, testing, and dozens more. Useful primarily as a discovery aid when starting a new project ("what's the current pick for X?") — the curation is opinionated (PRs adding obscure or unmaintained projects routinely get rejected), so each entry is at least credible. Not a tutorial, not API docs, just an annotated table of contents. The pattern is now everywhere ("awesome-cpp", "awesome-rust", "awesome-go", "awesome-hpp" — all derivative of this list's structure).
